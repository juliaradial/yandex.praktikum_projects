# Анализ убытков приложения PPRO+

**Цель исследования**

- найти причины убытков, которые терпит компания;
- проанализировать инвестиции на рекламу в разрезе региона, устройства и канала привлечения;
- найти пути выхода на прибыльность приложения.

**Результаты исследования**

Будут учтены при планировании расходов на рекламу на предстоящий период для избежания убытков.

**Навыки и инструменты**

Python,
Pandas,
Matplotlib,
когортный анализ,
юнит-экономика,
продуктовые метрики,
